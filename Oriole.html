The Origin Story

From a shared passion at Nchanga Secondary School to a global vision.

2023

A Meeting of Minds

In the halls of Nchanga Secondary School, Lusungu (14) and Alinaswe (15) connected over a deep passion for computer science and software architecture.

Early 2024

From Ideas to Prototypes

Discussions evolved into collaborative coding sessions. Lusungu began architecting FloatScript, while Alinaswe crafted the web interfaces to support it.

Mid 2025

The Birth of Oriole

Recognizing their complementary skills, they formalized their partnership, establishing Oriole—a name symbolizing innovation and the bright future of youth-led tech.

2025

Growth and Recognition

With FloatScript gaining traction, APYChat in development, and the recent launch of Trillium OS, Oriole secures its first round of seed funding.

Core Technologies

Pioneering tools built for performance and accessibility.

FloatScript

High-Performance Programming Language LIVE

A lightweight, high-performance language by Lusungu Luhana, designed for speed, readability, and versatility.

Official Website & Syntax Overview & Architecture How to Code: Tutorials

Aether

Universal ML Inference Engine LIVE

A powerful, lightweight inference engine by Lusungu Luhana that can be integrated into any codebase for machine learning purposes. Supports multiple frameworks and provides optimized performance for both training and deployment scenarios.

Technical Documentation Integration Guide API Reference

AXIS

Adaptive Experimental Intelligence System BETA

An advanced mini adaptive neural network system by Alinaswe Sinkala, capable of self-development and continuous learning. Built in Python, AXIS evolves its architecture based on data patterns and user interactions, offering personalized AI experiences.

Join Beta Waitlist

Trillium

Standalone Operating System LIVE

A revolutionary OS by Lusungu, recently launched, featuring a hybrid monolithic kernel with microkernel capabilities. It's designed for security, performance, and a minimalist user experience.

Technical Documentation Get Trillium OS Contribute (GitHub)

Regulus

Procedural Open World Game BETA

A sophisticated C++ game by Lusungu Luhana featuring auto world generation, vast open-world exploration, and dynamic environments. Experience infinite possibilities with procedurally generated landscapes and self-guided adventure gameplay.
