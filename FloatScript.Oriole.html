Simplicity First

FloatScript removes the ceremony of traditional languages, allowing you to focus on the logic. No semicolons, no curly braces, just clean, readable code.

lex count = 0
while count < 5
std::out count
count = count + 1

Power When Needed

Despite its simple syntax, FloatScript is built on a robust C++ core, giving you access to low-level performance and complex data structures when your project demands it.

func fib(int n)
  if n <= 1 return n
  return fib(n-1) + fib(n-2)

Learn FloatScript in 60 Seconds

Get started with our streamlined tutorial. Master the basics and see why FloatScript is the future of programming.

1

Variables & Types

FloatScript uses lex for variable declaration and is dynamically typed. No need to specify int, float, or string. It just works.

// A number variable
lex my_age = 14

// A string variable
lex my_name = "Lusungu"
2

The Print Statement

Our standard output function is std::out. It's simple, fast, and does what it says on the tin.

std::out "Hello, FloatScript!"

lex pi = 3.14159
std::out "The value of pi is: " + pi
3

Functions & Logic

Declare functions with func. The syntax is clean, and the logic is easy to follow.

// A simple greeting function
func greet(name)
    std::out "Hello, " + name + "!"

// A call to the function
greet("FloatScript")
