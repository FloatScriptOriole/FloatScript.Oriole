The Problem

C++ is powerful but overwhelming. Python is friendly but slow. Most languages sacrifice one for the other, leaving developers stuck in the middle.

The Gap

The Solution

FloatScript is lightweight and expressive with built‑in threading, JIT & AOT compilation, pointer support, and memory control — no compromises.

FloatScript

The Process

Define functions with fn, run async tasks with init, tap into seven built‑in libraries. Write once, run anywhere.

Batteries Included

Manual Memory Control

Low‑level memory access via #include <mem> — allocate, write, read, and free blocks directly. Ideal for performance‑critical workloads.

Low-Level Power

Classes & OOP

Define classes, navigate them with with and this, and manipulate members intuitively. Structured OOP in a scripting language.

Version History

From Zero to FloatScript

A language's journey, one release at a time.

OptScript

The original prototype. Where it all started — a 14‑year‑old's first interpreter, born from curiosity and a few hundred lines of C++.

FloatScript Beta

First public release. Introduced the lex keyword, basic functions, and standard I/O. Rough but real.
